# react-from-scratch

## What we're building

By the end of this workshop you'll have built an **interactive product card** for a fictional clothing brand — from a static HTML-like component all the way to something that fetches data and responds to user interaction.

More importantly, you'll understand **why** every line is there.

## What you'll learn

| Exercise | Concept |
|----------|---------|
| 01 | Static components — JSX, structure, returning UI |
| 02 | Props — making components reusable |
| 03 | State — making components interactive with `useState` |
| 04 | Effects — syncing with the outside world using `useEffect` |

## Getting started (no install needed)

Open the StackBlitz starter template: **[link coming soon]**

Fork it and follow along with each exercise in the `exercises/` folder.

If you prefer to work locally:

```bash
git clone https://github.com/TalenMud/react-from-scratch
cd react-from-scratch
npm install
npm run dev
```

## Structure

```
exercises/     — starter files with TODOs for each concept
solutions/     — full working solutions (spoilers!)
```

## Mini challenge

At the end of the workshop there's an open-ended challenge — take what you've built and make it your own. Add a feature, change the styling, build a new component. Solutions will be shared at the end of the stream.
