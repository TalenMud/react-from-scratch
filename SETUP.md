# Setup

## Option 1 — Browser (recommended for the workshop)

Open the StackBlitz template: **[link coming soon]**

No install needed. Just fork and start coding.

## Option 2 — Local development

You'll need [Node.js](https://nodejs.org/) installed (v18 or above).

```bash
# Clone the repo
git clone https://github.com/TalenMud/react-from-scratch
cd react-from-scratch

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open http://localhost:5173 in your browser.

## Working through the exercises

Each exercise lives in `exercises/XX-concept/`:
- Read the `README.md` first
- Open `starter.jsx` and fill in the TODOs
- Check `solutions/XX-concept/solution.jsx` if you get stuck

Work through them in order — each one builds on the last.
